Restaurant Sample Dashboard

See this interactive dashboard here

The Project:
Design and build an interactive dashboards for business owners to see all information easily

Objective:
Track KPIs (sales, #of guests, # of orders, PPA), flexibility to compare sales performances, # of guests and average order turnover based on the following parameters: time period, dine-in or take-out. Compare sales of each menu items per categories

Data Source:
Excel files from restaurant's POS system (Toast) containing information about order, sales, menu items

Steps Taken:
Connect and clean data on Excel
Connect to Tableau and analyze different components
Create custom calculations and parameters
Design interactive dashboard

*Some of the data has been hidden or modified for privacy
