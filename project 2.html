Maven Analytics Pizza Challenge

See the dashboard

The Project:
Create a single page dashboard and use data to improve operations

Objective:
Answer the following brief:
What days and times are the busiest?
How many pizzas are made during peak periods?
What are the best and worst selling pizzas?
What's the average order value?
How well is the seating capacity utilized?

Data Source:
Excel files containing orders detail, sales and menu items

Steps Taken:
Connect and clean data on Excel
Connect to Tableau and analyze different components
Create custom calculations and parameters
Design static dashboard

On top of answering the brief, I also found that more than half of the ingredients are only used on one type of pizze each.
This analysis is what ultimately put me in the final.
