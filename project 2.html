Maven Analytics Pizza Challenge

See the dashboard

As I was finishing up my Google Data Analytics Capstone Project, I saw that Maven Analytics came out with a new challenge which is the Pizza Challenge. With my experiences I feel this is right up my alley, so I decided to join in. On this challenge, I was playing the role of a BI Consultant hired by Plato's Pizza, a Greek-inspired pizza place in New Jersey. My task was to create a single page dashboard and use data to improve operations.
